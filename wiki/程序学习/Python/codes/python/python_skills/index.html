if name == ‘main‘:

  • 为什么有的执行代码前都要写一个if
  • __name__是Python中一个隐含的变量它代表了模块的名字
  • 只有被Python解释器直接执行的模块的名字才是__main__
  • 也即是，除非直接执行该代码，否则是不会执行if __name__ == '__main__':下面的代码的。
  • if __name__ == '__main__':下的变量是全局变量
